# novel_laomao

## 老猫小说APP爬虫

---
## 实现功能
<ul>
<li>支持多线程，默认为6线程</li>
<li>通过Bookid下载小说</li>
<li>通过书名下载小说</li>
<li>通过分类序号批量下载小说</li>
</ul>

### 环境需求

<ul>

<li>Python3.4或以上</li>

</ul>

</ul>
